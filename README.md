# Bank Management System

A Java Swing + MySQL based ATM simulation project.

## Features
- Login
- Deposit
- Withdraw
- Balance Enquiry
- Fast Cash
- Mini Statement
- PIN Change

## Technologies Used
- Java
- Swing
- JDBC
- MySQL

## Database Setup

1. Open MySQL
2. Create a database:
   CREATE DATABASE bankmanagementsystem;

3. Import the SQL file:
   database/Dump20260512.sql

## Tables Used
- signup
- login
- bank
